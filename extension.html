How to install extension

How to install the Browse in Darkness extension in a few easy steps*
1. Open the ZIP file that contains the extension and there will be a folder inside that contains the extension.
2. Go to the extensions page of your browser. For Chrome that is "chrome://extensions" and Brave is "brave://extensions".
3. Click on the "Developer Mode" switch.
4. Click on the "Load unpacked" button and select the file that contains the extension.
5. The extension should be ready and will be in the extensions bar with a small yellow flame icon.

*For other browsers besides Brave and Chrome these steps may be different.
